## DIY Art Resources

This repository contains a small collection of notes and references related to creative DIY art projects.

Paint by numbers is a popular creative hobby among adults, combining structure with relaxation and allowing people to create artwork step by step without prior experience.

A helpful overview of paint by numbers projects for adults can be found here:
https://lieblingsart.com/collections/malen-nach-zahlen-erwachsene

This repository is intended as a lightweight reference for anyone interested in creative hobbies and DIY art.
